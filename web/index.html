CSU Deans EP Tool

Public, deterministic re-derivation of the Earnings-Premium test for arts programs. Uses the Department's published data; cites every rule.

Read first. This tool is built by educators — not attorneys, not legislative analysts. Every number is a forward simulation on the AHEAD PPD:2026 negotiator analytic file; the first official STATS earnings-premium release is scheduled for 2027-07-01. Verify all findings against primary sources before any external submission, public comment, or institutional decision.

Ground rules

  • Public data only. The tool does not collect institutional records.
  • No accounts, no login. UNITID and CIP list are the entire input.
  • Every verdict cites the regulation, the statute, or the Department's preamble it rests on.
  • The verdict word — PASS, FAIL, NOT MEASURED — is operationally specific, not a color band.
